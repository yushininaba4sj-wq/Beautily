# Beautily

AI美容プロデューサーアプリ（Next.js + Tailwind CSS v4）。

顔写真から診断・美容カルテ・提案・シミュレーション・AI秘書・買い物同行・クローゼット・コスメ管理・美容院サポート・垢抜けロードマップまで一気通貫。

> 現状のAI分析・画像生成は**デモ（モック）**です。本番はVision / 画像生成APIに差し替え可能な構成です。

## 開発

```bash
cd Beautily
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## ビルド

```bash
npm run build
npm start
```
